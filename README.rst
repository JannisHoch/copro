conflict_model
----------------
(ML) model for mapping environmental drivers of conflict risk

execution
----------------
.. code-block:: console
  $ cd conflict_model
  $ sh run.sh

This automatically executes the ipynb and converts it to a html-file, also stored in the conflict_model folder.
